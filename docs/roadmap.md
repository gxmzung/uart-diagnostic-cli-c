# Roadmap

## v0.1

- UART-style log parsing
- INFO / WARN / ERROR / SENSOR detection
- sensor value extraction
- diagnostic summary report

## v0.2

- CLI argument support
- output report file
- malformed log detection
- warning severity levels

## v0.3

- binary UART frame parsing
- CRC validation
- ring-buffer style parser
- embedded test harness

## v1.0

A C-based UART diagnostic CLI for embedded field operation and device debugging practice.
